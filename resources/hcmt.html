general info

The newest train to run on the Melbourne network, the High-Capacity Metro Train (HCMT) has more standing room and new technogolies making them able to support a growing Melbourne population.

The introduction

Melbourne's population is growing faster than the Railways can keep up with, so the logical solution is new trains for the busiest Melbourne lines; These new trains need to be able to have increased capacity, and be accessible to everyone including those with physical disabilities.

The new trains

The High-Capacity merges all the good elements from all 2017 trains, Comeng's big windows, the Nexas full back chairs and articulated gangway, X'trapolis floor, roof, and wheelchair access; the new trains also include some new features such as fold-up seats where the wheelchairs can go.

The New Features

There is many new features on this train, which is set to be the new standards for Melbournes trains and have already been adapted to the new PTV refurbishment Comeng's such as real-time passenger info screens, real-time CCTV in HD and seat fabrics which show which seats are priority seating and which are not.

The High-Capacity also have more verticle handles, standing room, Passenger info screens which can be seen when the train is packed in peak hour and anti-vadalism seat backs.
